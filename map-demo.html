커스텀 핀 지도 데모

아래 예시는 OpenStreetMap 타일 + Leaflet(무료 오픈소스 라이브러리)입니다. 업체 계정 없이도 동작 확인이 가능하지만, 상용·대량 트래픽은 타일 정책을 따라야 하며 한국 서비스는 카카오 · 네이버 지도 SDK가 더 적합할 때가 많습니다.

행로 브랜드 색에 맞춘 핀 예시

SVG 모양 그대로 쓸 수 있고, 이미지(png/svg) URL로 교체하면 디자인팀 산출물을 그대로 적용할 수 있습니다. 핀을 누르면 해당 지점의 상세 주소 카드가 먼저 뜨고, 카드 안의 자세히 보기를 누르면 변경 전·후 소개 페이지로 이동합니다.

사용 흐름: 핀을 누르면 위치 카드(제목 · 주소 · 링크)가 먼저 열리고, 카드 안의 자세히 보기를 통해 place-detail.html?id=…로 이동합니다.

표시 좌표는 예시입니다(서울 시청 근처). 카카오/네이버로 옮길 때는 동일하게 위도(WGS84)에 핀 좌표를 넣습니다. 카카오 JavaScript API의 MarkerImage나 커스텀 오버레이로도 같은 방법이 가능합니다.
